About the Edit Mode Buttons
The buttons above allow you to control your session:
Manage Files Opens the File Manager where you can upload and organize files that are part of this publication. The File Manager will appear in a separate tab in your browser.
Preview Switches to Preview Mode to show you how the page will look to the public after publication. The editing controls and buttons will be hidden. When in preview mode, this button will change to "Edit"; click it again to return to Edit Mode.
Discard Discards all changes that have been made to this page in the current session (since it was opened). Use this button to abandon any changes you have made. You will have the opportunity to continue editing using the initial contents of the page.
Submit for Review Click this button when all edits are complete and you are ready to submit it for publication. Your draft will be analyzed for any missing data or recommended improvements that can be made; these will appear in a pop-up window where you can either confirm the submission or return to editing. (If any missing data is required, you will not be able to proceed with the submission.) Proceeding with the submission will usually send the draft through the NPS review process; however, if only minor changes are indicated, NPS review is not required and the draft will be submitted directly for publication.
